Una vez activado Esto bloqueara la publicidad en las aplicaciones y en algunas Web


Instalar la App Edge Browser

Este navegador permite instalar extensiones web (gracias al codigo de KiwiBrowser) y podes bloquear toda la publicidad de las paginas web. Deberas Instalar la Extension uBlock y Listo.
